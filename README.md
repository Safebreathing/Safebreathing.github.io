# AirSense – Indoor Air Quality Monitoring

**AirSense** is a web-based project that monitors and visualizes indoor air quality in real-time.  
It connects to a device that measures **CO₂ levels, oxygen, humidity, and overall air quality**, giving users clear insights to maintain a healthy environment.

## Features
- Real-time monitoring of CO₂, O₂, and humidity levels.
- Historical data tracking with trend visualization.
- Alerts for poor air quality conditions.
- Responsive and interactive web interface.
- Designed for **homes, schools, and offices**.

## Technology Stack
- **Frontend:** HTML, CSS (SASS), JavaScript

## Project Goal
To provide a **user-friendly solution** for monitoring indoor air quality, promoting health, focus, and productivity.
